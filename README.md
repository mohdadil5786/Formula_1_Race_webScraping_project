# 🏎️ Formula 1 Race Results Web Scraping & Data Analysis

This project showcases an end-to-end data analytics workflow by extracting Formula 1 race results from the official Formula 1 website using web scraping techniques. The collected data was transformed into a structured dataset, cleaned, and analyzed to uncover meaningful insights into driver and team performance throughout the racing season.

Using Python and industry-standard data analysis libraries, the project demonstrates the complete process of data collection, preprocessing, exploratory data analysis (EDA), and visualization. Various analytical techniques were applied to identify winning trends, team dominance, race statistics, and performance patterns.

### Key Highlights

* Scraped Formula 1 race results using Requests and BeautifulSoup.
* Converted raw HTML data into a structured Pandas DataFrame.
* Performed data cleaning and preprocessing for accurate analysis.
* Conducted exploratory data analysis to identify key performance trends.
* Created visualizations using Matplotlib and Seaborn to communicate insights effectively.
* Generated actionable insights on driver success rates, team performance, and race statistics.

### Technologies Used

* Python
* Requests
* BeautifulSoup
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

This project demonstrates practical skills in web scraping, data wrangling, exploratory data analysis, and data visualization while working with real-world sports data.
